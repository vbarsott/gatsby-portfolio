Gatsby Portfolio

- Install gatsby-cli (if not installed: npm i -g gatsby-cli)
- Create a new Gatsby site using the hello-world starter:
  gatsby new my-hello-world-starter https://github.com/gatsbyjs/gatsby-starter-hello-world
- gatsby-source-filesystem: npm install gatsby-source-filesystem
- gatsby-transformer-remark: npm install gatsby-transformer-remark
- gatsby-plugin-image: npm install gatsby-image
and
npm install gatsby-transformer-sharp gatsby-plugin-sharp
- To deploy the website to a host:
  - gatsby build
  - deploy using netlify (if )
