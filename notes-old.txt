Gatsby:
- npm i -g gatsby-cli
- Go to: https://www.gatsbyjs.com/
- Search for Gatsby CLI commands (https://www.gatsbyjs.com/docs/reference/gatsby-cli/#how-to-use-gatsby-cli)
- Copy: gatsby new my-awesome-blog-site https://github.com/gatsbyjs/gatsby-starter-blog
- Change the name my-awesome-blog-site to react-gatsby-web-warrior
- Run: gatsby develop (control c to kill)
- Open: http://localhost:8000/
- Go to: https://www.gatsbyjs.com/docs/how-to/plugins-and-themes/creating-a-source-plugin/#how-to-create-a-source-plugin
- Install: npm install gatsby-source-filesystem
